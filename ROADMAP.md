# 개발자 웹 이력서 개발 ROADMAP

## 프로젝트 개요
HTML, CSS, JavaScript, Tailwind CSS를 활용한 반응형 개발자 웹 이력서 개발

## 이력서 기본 컨텐츠 구성

### 개인 정보
- 이름, 직위/직군
- 프로필 사진
- 연락처 (이메일, 전화번호, GitHub, LinkedIn 등)
- 간단한 자기소개

### 경력 사항
- 회사명, 부서, 직위
- 근무 기간
- 주요 성과 및 책임

### 기술 스택
- 프로그래밍 언어
- 프론트엔드 기술
- 백엔드 기술
- 데이터베이스
- 도구 및 플랫폼

### 프로젝트 포트폴리오
- 프로젝트명
- 프로젝트 설명
- 사용 기술
- 링크 (GitHub, 라이브 데모)

### 학력
- 학교명
- 전공
- 졸업 연도

---

## 기술 스택 요구사항
- **HTML5**: 의미있는 마크업 구조
- **CSS3**: 반응형 디자인, 애니메이션
- **JavaScript (Vanilla)**: DOM 조작, 이벤트 핸들링
- **Tailwind CSS**: 유틸리티 기반 스타일링
